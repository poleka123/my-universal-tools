## 仓库结构
📂 01-my-scripts：个人编写的效率小工具
- 文件处理、文本工具、系统工具等

📂 02-collected-tools：网上整理的实用工具
- 标注来源 + 极简使用说明

📂 03-project-utils：项目通用工具类（开发复用）
- python/：Python 项目通用工具（时间、文件、字符串、网络请求）
- java/：Java 项目通用工具
- frontend/：前端通用工具函数

## 使用规范
1. 新增工具：放入对应分类文件夹，写简单注释
2. 同步更新：git 提交推送，跨设备随时拉取
3. 项目复用：直接复制工具类到新项目，快速开发
